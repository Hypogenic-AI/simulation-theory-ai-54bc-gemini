# Cloned Repositories

## 1. generative_agents
- **URL**: https://github.com/joonspk-research/generative_agents
- **Purpose**: Official implementation of "Interactive Simulacra of Human Behavior".
- **Key Files**: `reverie/backend_server/reverie.py` (Main simulation engine).

## 2. homo_silicus
- **URL**: https://github.com/johnjosephhorton/homo_silicus
- **Purpose**: Framework for using LLMs as simulated economic agents.
- **Key Files**: `experiments/` directory contains various behavioral tests.

## 3. AgentSociety
- **URL**: https://github.com/tsinghua-fib-lab/AgentSociety
- **Purpose**: Large-scale city-level simulation of LLM agents.
- **Key Files**: `agentsociety/` core framework.

## Usage Note
Each repository may have its own dependency requirements. It is recommended to create isolated environments for running their experiments.
